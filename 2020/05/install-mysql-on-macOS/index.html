MySQL在macOS上的安装

System version: macOS Catalina 10.15.5

MySQL cersion: 8.0.20

MySQL offical download site: https://dev.mysql.com/downloads/mysql/

1-安装

前往MySQL的官方网站下载安装MySQL Community Server（免费版）

选择DMG Archive或者Compressed TAR Archive均可。这里选择dmg格式的包。

下载，打开压缩包，双击运行pkg安装包。

如果出现“无法验证开发者”的情况，前往系统偏好设置->安全性与隐私->通用->仍要打开

之后按照提示下一步。

最后安装完成需要你设置root用户的密码，自行设置密码后牢记。

建议安装后选择自动启动MySQL Server服务，否则请按照启动与停止启动服务。

2-使用

如果你在Mac上的终端上输入mysql后提示command not found，那么请这样做：

  • 打开终端
  • 切换目录：cd /usr/local/bin
  • 设置软链：ln -fs /usr/local/mysql/bin/mysql mysql
  • 完成。只需要在终端中输入：mysql -u root -p即可使用MySQL。
