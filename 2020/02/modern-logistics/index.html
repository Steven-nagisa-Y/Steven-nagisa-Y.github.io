物流概念的产生

  • 发生在社会经济领域中的一种活动
  • 物质资料生产和流通活动的有机组成部分
  • 物流涵盖了从原材料的获取到半成品管理和产成品销售的全过程

物流的定义

不同的定义：

  1. 物流是从生产阶段到消费或利用阶段物资的移动及货物处理活动的管理。

  2. 物流是制品从生产地到最终消费者的物理性转移活动。具体是由包装、装卸、运输、保管以及信息等活动组成。

  3. 物流是将原材料、半成品以及产成品由生产地送达至消费地的所有流通活动。包括用户服务、需求预测、信息联系、物料搬运、订单处理、选址、采购、包装、运输、装卸、废弃物处理以及仓储管理。

  4. 物流是为满足消费者需求而进行的对原材料、半成品、最终产品及相关信息从起始地到消费地的有效流动与存储的计划、实施与控制的过程。

  5. 物流是对从生产阶段到消费或者利用阶段物资的移动及货物的处理活动的管理。

物流与营销的关系

  • 最早的物流定义是由营销组织提出来的。
  • 物流看作营销的要素之一。

物流和物流管理的关系

Logistics 与 PD的关系

二者的主要区别在于所涵盖范围的变化

Logistics是将物料采购供应、产品生产制造和产成品销售的存货流动依次对接。

运输和物流之间的关系

运输是物流活动的一个核心要素；

物流中其它要素的变化，都会对运输产生影响；

运输是现代物流服务的重要物质技术基础；

运输是社会物流体系的一个有机组成部分；

交通运输与社会生产的物流结构互相影响。

物流的作用

商品的价值

  1. 形式价值 => 产生于生产过程
  2. 时间价值 => 产生于存储过程
  3. 空间价值 => 产生于运输过程
  4. 所有权价值 => 产生于销售过程

作用

  1. 服务商流
  2. 保障生产
  3. 方便生活

效用

  1. 空间效用：供应链上游到下游，通过运输，使物品有空间的位移
  2. 时间效用：仓储设置弥补时间差异
  3. 品种效用：对订单进行处理，对物品进行整合
  4. 批量效用：生产商和供应商是批量的，末端是小批量的
  5. 信息效用
  6. 风险效用

隔离

  1. 社会：价值无法体现
  2. 空间：生产的场所和消费场所不同
  3. 时间：仓储
  4. 信息：厂商需要市场的消费和流通信息
  5. 功能：标准化的个性化的

物流活动的分类

按照发生的领域

  1. 微观
  2. 中观
  3. 宏观

按照在供应链中的组成部分

  1. 采购物流
  2. 工厂内物流
  3. 生产物流
  4. 销售物流
  5. 返回回收物流
  6. 再生利用和废弃物物流

按照地域

  1. 国内物流
  2. 国际物流
  3. 地区物流

按照系统性质

  1. 社会物流
  2. 行业物流
  3. 企业物流

按照从事物流的主体

  1. 第一方物流：提供方物流
  2. 第二方物流：采购方物流
  3. 第三方物流：物流服务提供商
  4. 第四方物流：物流服务集成商

  • 绿色物流
